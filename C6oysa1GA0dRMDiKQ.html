Agora, vou criar os arquivos separados para o CSS (styles.css) e JavaScript (script.js): styles.css: ```css body { background-color: #f0f0f0; } .card { box-shadow: 0 4px 8px rgba(0,0,0,0.1); } #display { font-size: 24px; text-align: right; padding: 10px; } .btn { font-size: 18px; font-weight: bold; } .btn-light:hover { background-color: #e0e0e0; } .btn-warning:hover { background-color: #ffc107; } .btn-success:hover { background-color: #28a745; } .btn-danger:hover { background-color: #dc3545; } ``` script.js: ```javascript let display = document.getElementById('display'); function appendToDisplay(value) { display.value += value; } function clearDisplay() { display.value = ''; } function calculate() { try { display.value = eval(display.value); } catch (error) { display.value = 'Erro'; } } // Adicionar suporte para entrada de teclado document.addEventListener('keydown', function(event) { const key = event.key; if (/[0-9+\-*/.=]/.test(key)) { event.preventDefault(); if (key === '=') { calculate(); } else { appendToDisplay(key); } } else if (key === 'Enter') { event.preventDefault(); calculate(); } else if (key === 'Backspace') { event.preventDefault(); display.value = display.value.slice(0, -1); } else if (key === 'Escape') { event.preventDefault(); clearDisplay(); } }); ``` Esta implementação cria uma calculadora interativa usando HTML, Bootstrap e JavaScript, com os arquivos separados conforme solicitado. O arquivo HTML contém a estrutura da página e os botões da calculadora, o arquivo CSS (styles.css) define o estilo da calculadora, e o arquivo JavaScript (script.js) contém a lógica para as operações da calculadora. A calculadora possui as seguintes funcionalidades: 1. Botões numéricos e de operações básicas (+, -, *, /) 2. Botão de igual (=) para calcular o resultado 3. Botão de limpar para resetar a calculadora 4. Suporte para entrada de teclado 5. Tratamento de erros básico Você pode personalizar ainda mais o design ou adicionar funcionalidades extras conforme necessário.
